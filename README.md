# matlab-demo
* 包含一些matlab函数的实现方法，方便在其他语言编写下实现相同功能，如unwrap的具体实现方法。
* 一些函数的使用方法，有些函数使用时可能忽略某些问题，编写后出现bug，这里总结了一些基本的调用方法。
* 编写一些实用的功能，但maltab没有封装函数，如画三维圆。
* 一些笔记等记录某些功能的使用技巧。
